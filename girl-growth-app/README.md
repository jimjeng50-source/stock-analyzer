# 美少女養成陪伴 App

現代都市背景的互動式養成陪伴 Web App。玩家扮演養育者，陪伴一位女兒從童年成長到成年——透過安排行程、日常對話與教養選擇，塑造她的能力、性格與人生，走向多種可能的結局。靈感來源：《美少女夢工廠》。

> ⚠️ 本專案處於**設計階段**。程式尚未開始撰寫；目前僅有設計文件。

## 一句話定位
- **核心樂趣**：劇情與情感陪伴
- **世界觀**：現代都市／繁體中文
- **平台**：網頁 Web App（前後端分離）
- **AI**：Claude（對話）＋ 生圖服務（女兒立繪）

## 內容界線
成長養育為主、清新健康；成年後可有健康的戀愛/情感結局。
**絕不製作任何性化／情色內容**（不論角色年齡設定），此為硬性底線。
